This repository contains the code for my personal portfolio, built primarily to prove to recruiters that I can ship a website without relying on a 500MB `node_modules` folder.

## 🛠 Why This Tech Stack?

Prepare to be underwhelmed. In a world obsessed with complex JavaScript frameworks, I chose violence by relying on absolute caveman technologies:

- 🧱 **HTML5 & CSS3:** The classic duo. Built without Tailwind CSS, because I enjoy the pain of specificity wars and writing actual CSS like it's 2012. 
- 📜 **Vanilla JavaScript:** Zero dependencies. No React, no Vue, no `npm install`. My `node_modules` folder is exactly 0 bytes. 
- 🤖 **Artificial Intelligence:** Let's be honest—while I forced the architecture back to the stone age, I pair-programmed the tricky bits with an AI assistant. I wrote the logic and structure, and the AI acted as a hyper-fast intern. We make a great team.

## 🧠 The Hinton Widget

If you look closely at the bottom left of the live site, you'll find a tiny, floating 🧠. Clicking it toggles a small panel that spits out deep, vaguely philosophical AI takes directly from the mind of Geoffrey Hinton. 

Why? Because every software engineer needs to artificially inject AI into their portfolio somehow to stay relevant. I just chose to make mine slightly unhinged.
